
## Setup Neovim
```
ln -sf ~/dotfiles/init.lua ~/.config/nvim/init.lua
ln -sf ~/dotfiles/lua ~/.config/nvim/lua
ln -s ~/dotfiles/colors ~/.config/nvim/colors
```

## Kitty Term
```
ln -sf ~/dotfiles/kitty.conf ~/.config/kitty/kitty.conf
```
